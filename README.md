# Fintrack - app de finanzas personales

Fintrack es un proyecto de una app para finzansas persoanesl, la idea es que se puedan llevar un registro, manejo, control de tus gastos e inversiones.
con opciones como poder conectarse a otras aplicaciones como nequi o bancolombia.

## Que tiene y tendra la app

### Que tiene
* inicio
* gastos
* cartera
* reportes
* perfil

### Que tendra
* llevar una cuenta de ingresos
* gastos
* metas
* inversiones - categorizados segun prioridad y clase.
* contara con opciones de transferir, analizar, metas y registro de gastos o costos;
    * en los gastos tambien se pueden ver los presupuestos.

## hacia que publico esta hecho la app
Hacia personas que quieran tener un control de su gastos y su dinero, la app no esta hecha para solucionar la vida financiera, pero puede ayudar en algunas cosas.

## Tecnologias.
* framework - expo
* app movil - expo go para poder ver el resultado
* IA - Claude pro para no perder la cabeza solucionando los errores
